## <해설>
ex. int x = 10일 경우
printf("%d", x++) -> x = 10
printf("%d", x++) -> x = 11
printf("%d", x) -> x = 11

둘 다 x = x+1 을 의미
x++ (후위 연산자, ++이 뒤에 있음) -> 식의 뒤에 있는 x값 -> 즉, 1을 더하기 전인 '10'
++x (전위 연산자, ++이 앞에 있음) -> 식의 앞에 있는 x값 -> 즉, 1을 더한 후인 '11'

but. 둘 중 뭐를 하든
printf("%d", x) -> 11
-> 전위, 후위 둘 다 결국엔 'x = x+1'이라는 식을 나타내기 때문
